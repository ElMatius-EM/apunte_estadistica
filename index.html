Intervalos de Confianza

Elige el parámetro que quieres estimar para ver la fórmula correcta y sus condiciones de uso.

Promedio / Media (μ)

¿Conoces la varianza poblacional (σ²)?

SÍ

Usa Normal (Z).

x̄ ± Z(α/2) * (σ / √n)
NO

Usa t-Student.

x̄ ± t(n-1, α/2) * (S / √n)

Nota: gl = n - 1.

Variabilidad / Varianza (σ²)

Usa Chi-Cuadrado (χ²).

Límite Inferior:

(n-1)S² / χ²derecho

Límite Superior:

(n-1)S² / χ²izquierdo

¡Intervalo no simétrico!

Porcentaje / Proporción (π)

Usa Normal (Z).

p̂ ± Z(α/2) * √[p̂(1-p̂)/n]

Nota: Calcula p̂ = éxitos / n.

Pruebas de Hipótesis Paramétricas

Tomando Decisiones con Datos para Media (μ) y Proporción (π)

  1. Plantear Hipótesis Nula (H₀) y Alternativa (H₁).
  2. Definir el nivel de significancia (α).
  3. Elegir el Estadístico de Prueba (Z, t).
  4. Establecer la Regla de Decisión (definir zona de rechazo).
  5. Calcular el valor del estadístico con los datos de la muestra.
  6. Concluir en términos del problema.

La Hipótesis Alternativa (H₁) te dice todo:

  • >
    Cola Derecha. Si H₁ usa "mayor que", la zona de rechazo está a la derecha.
  • <
    Cola Izquierda. Si H₁ usa "menor que", la zona de rechazo está a la izquierda.
  • ≠
    Dos Colas. Si H₁ usa "diferente de", la zona de rechazo se divide en ambos extremos. (¡Recuerda usar α/2!)

Pruebas de Hipótesis Chi-Cuadrado

χ² = Σ [ (Observado - Esperado)² / Esperado ]

Prueba de Bondad de Ajuste

"¿Mis datos siguen un patrón esperado?"

H₀: Los datos SÍ se ajustan a la distribución esperada.

gl = k - 1 - r

k = categorías, r = parámetros estimados

Prueba de Independencia

"¿Están relacionadas estas dos variables categóricas?"

H₀: Las variables son INDEPENDIENTES.

gl = (filas - 1) * (columnas - 1)

Las pruebas de Chi-Cuadrado son SIEMPRE de cola derecha.

Regresión y Correlación

Correlación (r)

Mide qué tan fuerte y en qué dirección se mueven juntas las variables. Su valor va de -1 (correlación negativa perfecta) a +1 (correlación positiva perfecta). Un valor cercano a 0 indica ausencia de relación lineal.

Regresión (ŷ = a + bx)

Ecuación para predecir un valor de Y a partir de un valor de X.

Interpretación de b: "Por cada unidad que aumenta X, se estima que Y cambia en b unidades."

Prueba de Hipótesis para b: H₀: β = 0 (la relación no es significativa).

Coeficiente de Determinación (R²)

Se calcula como R² = r².

Interpretación CLAVE: "El [R² en %] de la variación en Y es explicado por la variación en X a través del modelo de regresión."
